# Core Concepts

## NAME

Основные концепции Linux: файловая система, пользователи, права доступа, процессы и управление пакетами.

## SYNOPSIS

```
01-core-concepts/
├── 01-filesystem-hierarchy.md     # Структура ФС, команды навигации
├── 02-users-groups-permissions.md # Пользователи, группы, права (chmod, chown)
├── 03-processes-and-services.md   # Процессы, systemd, демоны
└── 04-package-management-overview.md # apt, dnf, pacman - управление ПО
```

## DESCRIPTION

Этот раздел охватывает фундаментальные концепции Linux, которые нужны для уверенной работы с системой. Вы узнаете как работает файловая система, как управлять пользователями и разрешениями, как работают процессы и сервисы, и как устанавливать/обновлять программное обеспечение.

**Практическое применение:**
- Ориентироваться в файловой системе и эффективно работать с файлами
- Управлять пользователями, группами и правами доступа
- Мониторить и управлять процессами и сервисами
- Устанавливать, обновлять и удалять программное обеспечение
- Решать типичные проблемы в повседневной работе

## CONTENTS

### 1. Filesystem Hierarchy
**[01-filesystem-hierarchy.md](01-filesystem-hierarchy.md)** | Type: **Reference + How-to**

Стандартная структура файловой системы Linux (FHS), команды для навигации, просмотра и управления файлами.

Содержит:
- Структура FHS директорий (/, /bin, /home, /etc, /var, /tmp, и т.д.)
- Команды навигации (pwd, cd, ls)
- Просмотр содержимого файлов (cat, head, tail, less)
- Управление файлами (touch, mkdir, cp, mv, rm)
- Поиск файлов (find, locate, grep)
- Информация о дисках (df, du, lsblk)
- Абсолютные и относительные пути
- Символические ссылки (ln -s)
- Скрытые файлы и типы файлов
- Решение типичных проблем

→ Прочитайте первым - это основа для всего остального

### 2. Users, Groups, and Permissions
**[02-users-groups-permissions.md](02-users-groups-permissions.md)** | Type: **Reference + How-to**

Управление пользователями, группами и правами доступа - ключевой аспект безопасности Linux.

Содержит:
- Файлы пользователей и групп (/etc/passwd, /etc/shadow, /etc/group)
- Команды для работы с пользователями (useradd, userdel, passwd)
- Команды для работы с группами (groupadd, groupmod, gpasswd)
- Права доступа (chmod) - числовой и текстовый формат
- Специальные права (setuid, setgid, sticky bit)
- Владелец файла (chown)
- sudo и su - выполнение команд от других пользователей
- Таблица прав для файлов и папок
- Решение проблем с доступом

→ Прочитайте после 01 - нужно понимать файлы для работы с правами

### 3. Processes and Services
**[03-processes-and-services.md](03-processes-and-services.md)** | Type: **Reference + How-to**

Управление процессами и сервисами - как взаимодействовать с запущенными программами.

Содержит:
- Просмотр процессов (ps, top, htop, pgrep, pidof)

- systemctl команды для работы с сервисами

- Демоны (daemons) - фоновые программы
- Создание собственных сервисов
- Решение проблем с процессами

→ Прочитайте после 02 - нужно понимать пользователей для управления процессами

## NOTES

- **Команды используют разные диствибутивы** — apt (Debian/Ubuntu), dnf (Fedora), pacman (Arch)
- **Все команды требуют практики** — читайте и одновременно практикуйте на своей системе
- **Права доступа критичны** — неправильные права могут сломать систему
- **sudo требует осторожности** — команды от root'а могут повредить систему
- **Логи - ваш друг** — при проблемах сначала смотрите journalctl
- **Разные дистрибутивы = разные команды** — но принципы одинаковые
