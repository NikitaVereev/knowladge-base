В этом уроке мы запустим веб-сервер **Nginx**, который будет отдавать вашу собственную HTML-страницу. Мы сделаем это, не устанавливая Nginx на сам компьютер.

## Шаг 1: Подготовка

Создайте пустую папку и внутри нее файл `index.html`.

```html
<!-- index.html -->
<h1>Hello from Docker!</h1>
<p>Это работает в контейнере.</p>
```

## Шаг 2: Запуск контейнера

Выполните команду в терминале (находясь в той же папке):

```bash
docker run -d \
  --name my-site \
  -p 8080:80 \
  -v $(pwd):/usr/share/nginx/html:ro \
  nginx:alpine
```

### Разбор команды:
*   `docker run`: Создать и запустить контейнер.
*   `-d` (**Detached**): Запустить в фоне (вернуть управление терминалу).
*   `--name my-site`: Дать контейнеру понятное имя.
*   `-p 8080:80`: **Проброс порта**.
    *   `8080`: Порт на вашем компьютере (localhost).
    *   `80`: Порт внутри контейнера (стандартный для Nginx).
*   `-v $(pwd):...:ro`: **Volume (Bind Mount)**.
    *   Мы берем текущую папку (`$(pwd)`) с вашим `index.html`.
    *   "Вставляем" её внутрь контейнера по пути `/usr/share/nginx/html` (оттуда Nginx берет файлы).
    *   `:ro`: Read-Only (контейнер не сможет случайно удалить ваш файл).
*   `nginx:alpine`: Имя образа. `alpine` — это сверхлегкая версия Linux (5 МБ).

## Шаг 3: Проверка

Откройте браузер по адресу: [http://localhost:8080](http://localhost:8080).
Вы должны увидеть ваш заголовок "Hello from Docker!".

**Эксперимент:**
1.  Откройте `index.html` в редакторе кода.
2.  Измените текст на "Hello World v2".
3.  Обновите страницу в браузере. Текст изменится мгновенно! (Потому что мы использовали Bind Mount).

## Шаг 4: Уборка

Контейнер продолжает работать в фоне. Чтобы остановить и удалить его:

```bash
docker rm -f my-site
```
*(Флаг `-f` принудительно останавливает и удаляет работающий контейнер).*
