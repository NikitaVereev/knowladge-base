# Настройка SSH для Ansible

Для работы Ansible необходим доступ к серверам по SSH, желательно без пароля (по ключам).

Информация находится в - [[tools/ssh/how-to/generate-keys|Настройка SSH ключей (Ed25519)]]


## Troubleshooting
Если получаете `Permission denied`:
1. Проверьте права на папку `.ssh` на сервере (должны быть `700`).
2. Проверьте права на файл `authorized_keys` (должны быть `600`).
3. Убедитесь, что вы подключаетесь правильным пользователем (`remote_user` в конфиге).
