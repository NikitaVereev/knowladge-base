## Жизненный цикл

| Команда | Описание |
| :--- | :--- |
| `vagrant init [box]` | Создать `Vagrantfile` в текущей директории. |
| `vagrant up` | Создать и запустить виртуальную машину. |
| `vagrant halt` | Выключить машину (ACPI shutdown). |
| `vagrant suspend` | Поставить на паузу (состояние сохраняется в RAM на диске). |
| `vagrant resume` | Вывести из паузы. |
| `vagrant destroy [-f]` | Удалить машину и все ее диски. |
| `vagrant reload` | Перезагрузить машину (применяет изменения Vagrantfile). |

## Работа с машиной

| Команда | Описание |
| :--- | :--- |
| `vagrant ssh [name]` | Подключиться по SSH. |
| `vagrant provision` | Запустить провижинеры на работающей машине. |
| `vagrant ssh-config` | Показать настройки SSH (IP, порт, ключ). |
| `vagrant port` | Показать проброшенные порты. |
| `vagrant status` | Показать статус машин в текущем проекте. |

## Управление образами (Boxes)

| Команда | Описание |
| :--- | :--- |
| `vagrant box list` | Показать список скачанных боксов. |
| `vagrant box add [name]` | Скачать новый бокс. |
| `vagrant box update` | Обновить бокс для текущей среды. |
| `vagrant box prune` | Удалить старые версии боксов. |
