## 1. Инициализация проекта

Создайте папку для проекта и сгенерируйте `Vagrantfile`.

```bash
mkdir my-project
cd my-project
# Создаст конфиг на базе Ubuntu 22.04
vagrant init ubuntu/jammy64
```

## 2. Запуск (Up)

Скачивает образ (если нет) и запускает ВМ.

```bash
vagrant up
```

## 3. Подключение (SSH)

Вход внутрь виртуальной машины.

```bash
vagrant ssh
```
Чтобы выйти, наберите `exit`.

## 4. Остановка работы

У вас есть три варианта завершения работы:

1.  **Suspend (Сон):** Сохраняет состояние RAM на диск. Мгновенный запуск, но занимает место.
    ```bash
    vagrant suspend
    ```
2.  **Halt (Выключение):** Полное выключение питания. Освобождает RAM.
    ```bash
    vagrant halt
    ```
3.  **Destroy (Уничтожение):** Удаляет виртуальную машину и все данные диска. `Vagrantfile` остается.
    ```bash
    vagrant destroy
    ```
