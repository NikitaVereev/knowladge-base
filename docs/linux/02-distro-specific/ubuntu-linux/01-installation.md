# Ubuntu Installation

## Overview

Установка Ubuntu Linux. Графический установщик делает это простым.

**Что вы узнаете:**
- Загрузка с USB и выбор языка
- Подключение к интернету
- Разбиение диска (или использование целого диска)
- Создание первого пользователя
- Первая загрузка и обновление

## Prerequisites

- USB флешка с Ubuntu ISO
- 20+ GB свободного места
- Доступ в интернет

## Установка

### 1. Загрузитесь с USB

Вставьте флешку и перезагрузитесь. Нажмите F12/Del/Esc чтобы выбрать загрузку с USB.

### 2. Выберите язык

Выберите язык установки и нажмите "Install Ubuntu".

### 3. Клавиатура

Выберите раскладку клавиатуры (обычно выбирается автоматически).

### 4. Обновления и ПО

Выберите:
- ✓ "Install third-party software"
- ✓ "Install security updates during installation"

### 5. Разбиение диска

**Вариант 1: Использовать весь диск (простой)**
- Выберите диск
- Нажмите "Install Now"

**Вариант 2: Manual (если нужно разбить)**
- Создайте разделы:
  - 500 MB EFI
  - Остальное для root

### 6. Часовой пояс

Выберите ваш часовой пояс (обычно автоматически по IP).

### 7. Учётная запись

```
Username:     ваше_имя
Password:     сильный_пароль
Hostname:     имя_компьютера (ubuntu по умолчанию)
```

### 8. Завершение

Подождите пока скопируются файлы (~10 минут).

## После первой загрузки

```bash
sudo apt update
sudo apt upgrade -y
sudo reboot
```

## Key Takeaways

- Ubuntu установка очень простая
- Графический установщик делает всё за вас
- LTS версии (24.04, 22.04) рекомендуются
