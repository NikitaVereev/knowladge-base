# Выбор дистрибутива

## Три главных семейства

### Debian: стабильность

**Пакетный менеджер:** apt

**Популярные:**
- Ubuntu 24.04 LTS
- Linux Mint
- Pop!_OS

**Характеристики:**
- Консервативный подход
- Стабильность над всем
- Огромное сообщество
- Много информации в сети

### Red Hat: инновация

**Пакетный менеджер:** dnf

**Популярные:**
- Fedora 41 (новые технологии)
- Rocky Linux (RHEL совместимость)
- AlmaLinux (CentOS замена)

**Характеристики:**
- Быстрые обновления
- Новые технологии раньше
- Корпоративная поддержка

### Arch: контроль

**Пакетный менеджер:** pacman

**Популярные:**
- Arch Linux (полный DIY)
- Manjaro (Arch + удобство)

**Характеристики:**
- Полный контроль
- Rolling release (всегда свежее)
- Требует знаний
- Отличная документация (Arch Wiki)

---

## LTS vs Rolling Release

### LTS (Long-Term Support)

**Пример:** Ubuntu 24.04 LTS, Debian

- Новая версия 1 раз в 1-2 года
- Поддержка 5-10 лет
- Только security обновления
- ✅ Стабильно, предсказуемо

### Rolling Release

**Пример:** Fedora, Arch

- Постоянные обновления
- Всегда последнее ПО
- ⚠️ Может быть нестабильно

## Как выбрать?

| Хотите | Выбирайте |
|-------|-----------|
| Стабильность | Ubuntu LTS, Debian |
| Свежесть | Fedora, Arch |
| Контроль | Arch, Gentoo |
| Простоту | Ubuntu, Mint |
| Сервер | Ubuntu Server, Debian, Rocky |

## Важные команды для выбора

**Узнать свой дистрибутив:**
```bash
cat /etc/os-release
lsb_release -a
```

**Узнать версию kernel:**
```bash
uname -r
```

**Информация о системе:**
```bash
neofetch
screenfetch
```

## Проблемы и решения

### Не знаю какой выбрать

Начните с **Ubuntu 24.04 LTS**. Это популярный выбор, огромное сообщество, легко найти решения.

### Хочу попробовать несколько

Используйте **VirtualBox**. Установите несколько дистро в виртуалках, опробуйте.

### Своего дистро нет здесь

Определите к какому семейству он относится (Debian-based, RHEL-based, Arch-based) и используйте инструкции для него.
