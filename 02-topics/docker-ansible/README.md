### Docker + Ansible

Компактный справочник по Docker, Ansible и оркестрации. Файлы организованы как набор самостоятельных тем и мини-шпаргалок.

#### Структура

- [01-foundations/](01-foundations/README.md) — мотивация, сценарии использования, общая архитектура Docker + Ansible.
- [02-environment/](02-environment/README.md) — виртуальные машины, базовый Linux, SSH, установка Docker, Ansible и Vagrant.
- [03-docker-basics/](03-docker-basics/README.md) — контейнеры, архитектура Docker, жизненный цикл контейнера, базовые команды, логи и отладка.
- [04-docker-images-dockerfile/](04-docker-images-dockerfile/README.md) — образы, слои, Dockerfile, сборка и оптимизация образов.
